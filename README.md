# OLA Business Insights Dashboard

## 📌 Project Aim

The **OLA Business Insights Dashboard** is an end-to-end Business Intelligence project that transforms a raw, unstructured ride-booking dataset into an interactive Power BI dashboard. The project demonstrates the complete data analytics workflow, including data cleaning, SQL data management, KPI development, and visualization to generate actionable business insights for operational decision-making.

## 🚀 Project Objectives

* Convert an unsorted and inconsistent ride-booking dataset into a clean, structured, and analysis-ready dataset.
* Perform data preprocessing using Microsoft Excel and SQL Server Management Studio (SSMS).
* Develop meaningful KPIs and interactive dashboards in Power BI to monitor ride performance, revenue, customer satisfaction, and booking trends.
* Enable stakeholders to identify business patterns, improve operational efficiency, and support data-driven decision-making.

## 🛠️ Project Workflow

### 1. Data Preparation in Microsoft Excel

* Sorted the raw dataset based on the **Date** column.
* Standardized date and time formats.
* Verified data consistency before importing into SQL Server.

### 2. Data Cleaning using SQL Server Management Studio (SSMS)

* Imported the cleaned dataset into Microsoft SQL Server.
* Performed exploratory data analysis to understand the dataset.
* Corrected inconsistent **Date** and **Time** formats.
* Handled missing values using appropriate statistical methods (Mean/Mode).
* Removed leading and trailing blank spaces using TRIM functions.
* Eliminated unnecessary special characters from text columns.
* Removed duplicate and unwanted unique identifier columns.
* Standardized categorical values for meaningful reporting.
* Prepared a clean, analysis-ready dataset for Power BI.

### 3. Dashboard Development in Power BI

* Connected Power BI to the cleaned SQL dataset.
* Created DAX Measures for business KPIs.
* Designed an interactive dashboard with filters and slicers.
* Added vehicle category images for enhanced visualization.
* Implemented dynamic KPI cards and business metrics.

## 📊 Dashboard Features

* **7 Business KPIs**

  * Total Bookings
  * Total Revenue
  * Completed Rides
  * Cancelled Rides
  * Incomplete Rides
  * Average Customer Rating
  * Average Driver Rating

* **Visualizations**

  * Ride Status Distribution (Completed, Cancelled, Incomplete) – Pie Charts
  * Monthly Booking Trend – Area Chart
  * Revenue by Vehicle Type – Stacked Column Chart
  * Booking Analysis by Vehicle Category
  * Customer & Driver Rating Analysis
  * Interactive Filters and Slicers for dynamic reporting
